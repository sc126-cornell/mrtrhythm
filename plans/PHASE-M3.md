## 1. 目標（Plan）

1. **點車站看板**：任一車站點開，顯示雙向未來班次（終點、時刻、倒數）；收班顯示狀態。
2. **跟車模式**：跟隨鏡頭鎖定＋自由拖曳解鎖＋「回到列車」恢復；資訊卡升級（即時時速、下一站倒數）。
3. **搜尋**：站名中英模糊搜尋（「北車」「Taipei Main」可命中），跳轉並開看板。
4. **深連結**：視角／時間／倍速／跟隨或看板狀態編碼進 URL hash，開啟即還原。

**可展示物**：完整可玩的產品體驗——找站、看板、跟車、分享連結。

## 2. 工作項目（Do）

- [ ] T3.1 車站看板：Schedule station 發車索引（每站排序陣列＋二分）、面板 UI（桌機側欄／手機底板）、收班狀態、站點點擊（抑制底下列車誤選）
- [ ] T3.2 跟車模式：follow 狀態機（off／lock／free）、dragstart 解鎖、「回到列車」、時速計（模擬時間距離微分＋EMA 平滑）、資訊卡按鈕列
- [ ] T3.3 站名搜尋：zh/en/ID 子字串索引、下拉結果、跳轉開板、Esc 關閉
- [ ] T3.4 URL 深連結：hash 編碼（c/z/t/spd/f/s）、replaceState 防歷史轟炸、時間僅在「非現在」時編碼、載入還原
