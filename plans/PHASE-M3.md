1. **點車站看板**：任一車站點開，顯示雙向未來班次（終點、時刻、倒數）；收班顯示狀態。

## 5. Check 紀錄（2026-07-18 填寫）

### Code Review
- 範圍：ui/{stationboard,search,deeplink,controls}、basemap 站點回呼、schedule 發車索引、main 跟隨狀態機。ESLint／tsc 零發現。
- 發現與處置：
  1. **debounce 飢餓 bug（本 phase 唯一實質 bug）**：跟隨中 `panTo` 每幀觸發 moveend，deep link 的 debounce 計時器永遠被重置、hash 永不更新——改為 throttle（每秒至多一寫＋尾端保底）。教訓：高頻事件源上不可用純 debounce。
  2. 站點與地圖 click 來自同一 DOM 事件，用 150ms 時間戳 guard 防「點站誤選底下列車」——已註解，未來若加觸控長按需重審。
  3. 搜尋為線性子字串掃描（119 筆站名）——規模下毫秒級，不建索引。
  4. SPA 特性：hash-only 導航不重新 boot（深連結需整頁載入）——屬預期行為，分享情境皆為新開頁。
- 測試方法備忘：驗證移動中列車的點擊要先暫停時鐘（移動靶 + 截圖延遲會落空）。
