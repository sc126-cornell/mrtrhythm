# TASKS — 捷奏・北捷即時地圖 任務拆解

| 項目 | 內容 |
|---|---|
| 版本 | v1.0（對應 PRD/SDD v1.0） |
| 日期 | 2026-07-18 |
| 估時單位 | 人日（一人專注一天）；總計 **≈ 32 人日** |
| 標記 | 🔴 關鍵路徑　⚠️ 高風險先做　👤 需用戶本人操作 |

**執行策略：垂直切片。** M1–M3 期間一律先以「淡水信義線」單線端到端跑通（資料→地圖→模擬→看板→跟車），驗證整條技術鏈後才鋪全網。每個里程碑結束有一個「可展示物」。

**工作方式（PDCA，2026-07-18 與用戶約定）**：每個 phase（＝里程碑 M0–M5，共 6 個）開工前依 [plans/_TEMPLATE.md](plans/_TEMPLATE.md) 建立 `plans/PHASE-Mx.md` 計畫檔（目標、工作項、Verify、Validate）。Phase 完成後必經三關——**code review → verify（對照 SDD 的技術驗證）→ validate（對照 PRD 的需求驗收）**——結果寫回計畫檔，向用戶回報並取得確認後，才建立下一個 phase 的計畫檔開工。
